SmartCrawler Pro는 고급 웹 크롤링 기능을 제공하는 전문 데스크탑 애플리케이션입니다.

주요 기능:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔍 고급 크롤링 기능
• 정적/동적 웹사이트 지원 (requests + Selenium)
• 여러 페이지 자동 순회 크롤링
• 사이트별 특화 크롤링 (네이버 쇼핑, 부동산 등)
• 봇 차단 방지 기능 (간격 설정, 재시도 로직)

📊 데이터 관리
• 실시간 결과 테이블 미리보기
• 엑셀(xlsx) 파일로 결과 저장
• 제목, 가격, 이미지, 날짜 등 선택적 추출
• 에러 처리 및 복구 기능

🖥️ 사용자 친화적 인터페이스
• 직관적인 GUI 인터페이스
• 진행률 표시 및 실시간 상태 확인
• 크롤링 중지/재시작 기능
• 사이트 추천 및 기술스택 가이드 포함

시스템 요구사항:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

• 운영체제: Windows 7 이상 (64비트 권장)
• 메모리: 최소 4GB RAM (8GB 권장)
• 저장공간: 200MB 이상의 여유 공간
• 인터넷 연결: Selenium WebDriver 자동 다운로드용

설치 시 주의사항:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 백신 프로그램 예외 설정
   - 일부 백신에서 오탐지할 수 있습니다
   - 설치 폴더를 백신 예외 목록에 추가하세요

2. 관리자 권한
   - 일반 사용자 권한으로도 설치 가능합니다
   - Program Files 폴더 설치 시 관리자 권한 필요

3. 이전 버전 제거
   - 이전 버전이 있으면 자동으로 제거됩니다
   - 설정 파일은 보존됩니다

4. 방화벽 설정
   - Chrome WebDriver 다운로드를 위해 인터넷 접근 허용
   - 크롤링 시 외부 사이트 접근 허용

설치 후 첫 실행:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 첫 실행 시 Chrome WebDriver가 자동 다운로드됩니다
2. "사이트 추천" 탭에서 연습용 사이트를 확인하세요
3. "기술스택 가이드" 탭에서 크롤링 난이도를 확인하세요
4. 간단한 사이트부터 시작해보세요 (예: quotes.toscrape.com)

법적 준수사항:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

• robots.txt 파일 확인 및 준수

• 개인정보 보호법 준수 (개인정보 수집 금지)

• 상업적 이용 시 해당 사이트 정책 확인
