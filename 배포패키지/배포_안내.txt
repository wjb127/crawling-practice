SmartCrawler Pro 배포 패키지 준비 안내
========================================

이 폴더는 최종 사용자에게 배포할 파일들을 포함하고 있습니다.

📁 배포패키지/ 폴더 구성
------------------------

✅ README.txt - 설치 방법 안내
✅ 사용설명서.pdf - 상세한 사용법 매뉴얼  
✅ 라이선스.txt - 소프트웨어 라이선스 계약서
⚠️  setup.exe - Windows 인스톨러 (실제 파일 필요)

실제 배포를 위한 준비 단계
--------------------------

1. setup.exe 파일 생성:
   - Windows 환경 필요
   - Inno Setup 6.0 이상 설치
   - 프로젝트 루트에서 다음 명령 실행:
     > build_installer.bat
   - 생성된 dist/SmartCrawler_Pro_Setup.exe를 setup.exe로 이름 변경

2. 최종 점검:
   - 모든 파일이 존재하는지 확인
   - 파일 크기와 내용 검증
   - 테스트 설치 진행

3. 배포 방법:
   - ZIP 파일로 압축하여 배포
   - 웹사이트 다운로드 링크 제공
   - USB나 CD/DVD로 물리적 배포

추가 파일 (필요시)
------------------
- 체크섬 파일 (무결성 검증용)
- 릴리즈 노트 (버전별 변경사항)
- 시스템 요구사항 상세 문서
